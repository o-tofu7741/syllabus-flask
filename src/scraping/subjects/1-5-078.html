授業科目の概要
・Python を用いて基本的なプログラミングの知識やデータ処理を行うための知識を学習するとともに、演習を通じて実践力を身につける
・前半では主に Python の基本的な文法を説明し、後半では主に Python を用いた情報処理について説明する。
・毎回の講義後にプログラミングの演習課題を出す
・プログラミング環境は Web サービス (Google Colab) を使用する
本授業の到達目標
1．プログラミングに関する基本的な知識を理解し、説明できる。(知識・理解・表現) /（学習・教育到達目標c、重要度60%)
2．プログラミング言語を用いて基本的な情報処理を行うことができる。(技能・技術) /（学習・教育到達目標d-2、重要度40%)

成績評価基準
学生の到達目標１、２に対応して、課題や演習の実施状況および採点結果から評価する。
評価の方法・総合評価割合
・授業内容をまとめた総合演習を前半と後半の 2 回実施する
・総合演習: 40%、その他の演習問題・小テスト・レポート等: 60%を目安に評価する
テキスト等、学修のために購入・準備が必要なもの
【準備必須】
授業中にパソコンを使用して課題を実施するので用意をすること。 事前・事後の学習に関してもパソコンが必要である。
ハードウェア：　Windows パソコンまたは Mac 必須 (OS がサポート中のもの)。 スマホや iPad では操作性に問題があるなど、うまく動作しないことがある。
ソフトウェア：　プログラミング環境は Web サービス (Google Colab) を使用する。 大学の Google アカウントを使用すること。 Web ブラウザは標準でインストールされている Edge や Safari で良いが、最新のバージョンにアップデートしておくこと。
履修上の注意事項、事前・事後学修に必要な目安時間、フィードバックの方法、学修上の助言
【履修上の注意事項】
・毎回の講義後に出される課題は、基本的に 1 週間程度の期間で提出を締め切る。 遅延提出は減点対象とする。
・適宜、課題の解答を公開するため、締切を守らないとその課題が 0 点になることがある。
【事前・事後学修に必要な目安時間】
・毎授業回につき240分程度を目安として、授業の振り返りと次の学習範囲の予習をおこなうこと。(講義資料などを事前に配布する)
