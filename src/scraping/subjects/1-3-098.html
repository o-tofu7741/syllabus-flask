授業科目の概要
This course focuses on developing productive skills (speaking and writing), by using global issues as core topics. The main goal of this course is to equip students with the ability to share their ideas and knowledge in the hope of leading to discussions surrounding global issues. By doing this, students should gain confidence in their speaking and writing abilities, as well as reading comprehension skills. This course also offers students an opportunity to develop their learner autonomy through researching the topics discussed in class and sharing with their peers.
本授業の到達目標
Public speaking:
To be able to give a presentation at a length determined by your teacher
To be able to ask and answer questions
To be able to give basic reasoning
To be able to asks and answer open questions

Role-play / Discussions
To be able to discuss a topic in a group
To be able to answer questions about your topic
To be able to share your opinion / argument in a group

Writing:
To be able to write a paragraph
To be able to appropriately link ideas
To be able to understand a basic essay structure

Reading:
To be able to paraphrase
To be able to summarise ideas from a text
