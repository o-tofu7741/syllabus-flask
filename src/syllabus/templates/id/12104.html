授業科目の概要
青森県で3泊4日の研修旅行を行う（八戸市、三戸町、十和田市、弘前市、五所川原市、中泊町、外ヶ浜町、青森市）。
研修旅行に先立って、学生がそれぞれ研究計画を立て、研修パンフレットを作成する。
しおり作成を通じて、日本の各地域の特色や多様性を発見し、言語化する力を鍛える。
訪問した地域のなかで、外国との接点をも見つけ、日本文化と比較できる眼力を養う（DP9）。
日程は2023年8月下旬から9月上旬の予定で調整。
本授業の到達目標
ＤＰ９にあるように、日本の多彩な文化を知る。研修旅行の計画を自分たちでたてることにより、研修旅行に関する基本的な事項を了解する。また、宿泊を伴う旅行が困難な場合は、日帰りの研修を2日に渡って行う。

成績評価基準
研修旅行への参加が必須となる。地域文化に関する調査の基礎的能力を身につける。自らの問題関心をもって研究テーマを設定し、現地踏査を行う方法を身につける。調べた内容を自分で文書にまとめる力をつける。
最終的な成績評価は、90点以上でS、80点以上でA、70点以上でB、60点以上でCとする。
評価の方法・総合評価割合
研修旅行への不参加は不合格となる。
事前準備学習（しおり作成）60％、研修旅行中の学習状況30％、レポート10％。
テキスト等、学修のために購入・準備が必要なもの
