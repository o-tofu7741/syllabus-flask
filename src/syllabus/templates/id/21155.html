授業科目の概要
コンピュータゲームの制作を通して、利用者と人工物の相互的なやりとりである「インタラクション」をデザインする方法を学ぶ。その中で、利用者が情報を受け取ったり入力するための仕組みである「ユーザーインターフェース」や、製品やサービスを通して得られるユーザー体験である「ユーザーエクスペリエンス」について学ぶ。最終課題ではMake Code Arcadeを用いて「明星デザインを紹介するゲーム」を制作しポータブルゲーム機で動作させる。コンピュータプログラミングや電子工作の手法を学びながら、デジタル技術をデザイン分野に統合するための視点を学ぶとともに、情報通信技術を用いたコミュニケーション能力を高める。また上記課題に加えて、インタラクションデザインに関連する事例の調査課題に取り組む。
これらを通して、企画やアイデアを資料や具体案として美的に構成する力や、デザインの活動に必要な幅広い知識と教養、様々な社会に最適な「ヒト・コト・モノ」の関係性を創造的に思考し判断する力を養う。
本授業の到達目標
インタラクションを理解しデザインに応用できる
必要に応じたユーザーインターフェースをデザインできる
ユーザーエクスペリエンスを理解しデザインに応用できる
コンピュータプログラミングに必要な基礎的な技能を身につけている　
電子工作に必要な基礎的な技能を身につけている

【事前・事後学修に必要な目安時間】
毎授業回につき480分程度を目安として、授業の振り返りと次の学習範囲の予習（課題制作など）を行うこと。

【フィードバックの方法】
レポートや課題のフィードバックは必要に応じて授業内で行います。

【学修上の助言】
「デザイン基礎演習」（視覚デザイン基礎演習）の単位を取得していることが履修条件となります。
課題制作等で配布・貸し出しされた材料や道具以外に必要なものが出た場合は各自で用意する必要があります。
質問等がある場合は、内容を明瞭にして教員から指示のあった方法（メールや LMSの個別指導コレクション等）で連絡を行うこと。
